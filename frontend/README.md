# HealthGuard AI - Command Center Frontend

React-based command center dashboard for healthcare coordinators.

## Features

- Real-time hospital capacity monitoring
- Emergency case tracking
- AI decision review interface
- Interactive maps with ambulance tracking
- Alert management

## Technology Stack

- React 18
- TypeScript
- Vite
- TanStack Query (React Query)
- Tailwind CSS
- shadcn/ui components
- Mapbox GL for maps
- WebSocket for real-time updates

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
